# saturnpixies
The goal is to make a simple game using the PixiJS API.
The guidelines are:
  - Simplicity: I'm not trying to do anything groundbreaking gameplay-wise.
  - Value: We will not use anything else than the api so the code is built from the ground up.
  - Polish: I want to put out something decent and polished to demonstrate the power of the api
  - Accessibility: The game must be fast to boot up, playable in browser, end goal is to make it available online for my portfolio.
  - Experimentation: I'm learning as I'm making this so I wanna experiment a lot with the api.
